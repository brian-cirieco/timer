# Timer

This is a timer that is precise at 100ms intervals.

## Instructions

To install libraries:
`npm install`

To run tests:
`npm run test`

To deploy local app
`npm run start`

Alternatively, the app is deployed on GitHub Pages at https://brian-cirieco.github.io/timer/

## Tools Used

- Next.js
- TypeScript
- React.js
- Jest.js
- Material UI
- react-canvas-confetti
