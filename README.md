# Timer

This is a timer that is precise at 100ms intervals.

## Instructions

To install libraries:
`npm install`

To run tests:
`npm run test`

To deploy local app
`npm run start`

Alternatively, the app is deployed on Vercel at https://vercel.com/brian-ciriecos-projects/timer

## Tools Used

- Next.js
- TypeScript
- React.js
- Jest.js
- Material UI
- react-canvas-confetti
